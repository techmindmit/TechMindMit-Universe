🌍 TechMindMit Universe V10.0

TechMindMit Universe is a futuristic, browser-based 3D simulation that allows users to interact with a hyper-realistic Earth and Moon system using real-time AI Hand Gestures. Built with cutting-edge web technologies, this project redefines human-computer interaction by eliminating the need for a physical mouse or keyboard.
✨ Key Features

    Hyper-Realistic Earth & Moon: High-fidelity textures including night lights, terrain bump mapping, and specular reflections on oceans.

    Cinematic Cosmos: A deeply immersive universe featuring the Milky Way, multi-layered nebula gas clouds, and dynamic interstellar stardust.

    AI Gesture Navigation: Real-time tracking of 21 hand landmarks for high-precision control and interaction.

    Dynamic Animations: * Supernova Pulse: Triggered by a closed fist gesture.

        Hyper-Speed Spin: Activated via a V-Sign (Peace) gesture.

        Zoom Lock: Controlled through a pinch gesture.

    Futuristic HUD: A professional glassmorphism overlay providing real-time system metrics and status updates.

===>Gesture,Action,Visual Effect<===
Open Palm 🖐️,Spatial Rotation,Rotates the universe based on hand movement
Closed Fist ✊,Supernova Pulse,Expands a glowing energy ring from the Earth
V-Sign ✌️,Hyper-Speed,Rapidly accelerates planetary and cosmic rotation
Pinch 👌,Zoom Control,Smoothly zooms the camera into the Earth's surface

🛠️ Technology Stack

    Three.js: Primary 3D engine for WebGL rendering and planetary physics.

    MediaPipe: Advanced computer vision library for hand gesture recognition.

    JavaScript (ES6+): Core logic for orbital mechanics and gesture mapping.

    HTML5/CSS3: Futuristic UI design with responsive layouts.
